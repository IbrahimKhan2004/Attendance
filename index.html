Fundamentals of Python Programming

Introduction to Python: Features, history, and applications. Installation and Python IDEs: IDLE, Jupyter Notebook. Python syntax, indentation, and comment. Input/output using input(), print(), Type casting and expressions.

Conditional statements: if, if-else, elif, nested conditions. Looping constructs: for, while, range(). Loop control statements: break, continue, pass, else with loops Introduction to functions: def, parameters, return values.

Variable scope: local, global, Introduction to modular programming and code reusability.
